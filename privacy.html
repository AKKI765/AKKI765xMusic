Access to Local Files

To enhance your experience, our website requires access to a specific folder on your device which contain your playlist and in that playlist your mp3 songs will be present.

Select only directory folder which contain Playlists. i.e,"Music/Playlists/...mp3" Choose Music folder only.

Add cover.jpg and info.json for "title" and "description"

If you do not give any image or json file then it will automatically take the default things.

  • Example: "Music/Playlists/...mp3"

Within this directory, you may organize your music as follows:

  • Playlist Folders: "Music/Playlists/...mp3" /Playlists/ representing playlists.
  • MP3 Files: Each playlist folder contains MP3 files.
  • Playlist Information: Each playlist folder includes a info.json file containing the title and description of the playlist.
  • Playlist Cover Image: Each playlist folder includes a cover.jpg image representing the playlist cover.
